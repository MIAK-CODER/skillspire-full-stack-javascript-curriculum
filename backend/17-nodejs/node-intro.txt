Macbooks
	- Install homebrew
		- Go to https://brew.sh/
		- Run this command:
			- /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
	- Run this command:
		- brew install node
	- Verify that brew and npm have been installed by running the following:
		- node -v
		- npm -v

What is Node.js?

- Node.js is a server side framework for JavaScript
- Can be used to make HTTP requests to an API
- Can be used to modify data in db

What is NPM?

- Node Package Manager allows you to install and manage packages
